# Amazon Prime Video Data Analysis

## Project Overview
An Exploratory Data Analysis (EDA) of 9,000+ Amazon Prime titles using Python. [cite_start]This project dissects the platform's schema and identifies content trends[cite: 209, 219].

## Product Dissection
* [cite_start]**Platform:** Amazon Prime Video[cite: 210].
* [cite_start]**Real-World Problem:** Solving content discovery and digital disconnect through visual storytelling and personalized recommendations[cite: 91, 107].
* [cite_start]**Schema Design:** A relational model linking `Titles` and `Credits` via a unique `id`[cite: 58, 148].

## Visualizations & Insights
This project includes **20 mandatory visualizations** covering:
1. Content Type Distribution (Movies vs. Shows)
2. Genre Popularity
3. IMDb vs. TMDB Score Correlations
4. Global Production Trends ... and 16 others.
